# 🏢 공공기관 소관부서 및 사무분장 법령 통합 검증 엔진 (웹 버전)

본 프로그램은 기존 Python 데스크톱 버전을 기반으로 누구나 웹 브라우저에서 편리하게 접속하여 사용할 수 있도록 HTML, CSS, JavaScript의 **3종 파일**로 새롭게 구성되었습니다. 별도의 실행 파일 다운로드나 라이브러리 설치 없이 **GitHub Pages**를 통해 누구나 웹 링크만으로 접속하여 사용할 수 있습니다.

---

## 📂 파일 구성 (3종 파일)

1. **`index.html`**: 현대적인 레이아웃과 접근성 높은 대시보드 뼈대를 제공하는 마크업 문서
2. **`style.css`**: 글래스모피즘(Glassmorphism)과 반응형 그리드, 다크/라이트 모드를 완벽 지원하는 세련된 스타일시트
3. **`app.js`**: 국가법령정보 API 호출, CORS 우회 로테이션 프록시, 소관 부서 역추적 추출 알고리즘, 결과 내 검색, SheetJS 기반의 엑셀 다운로드 비즈니스 로직

---

## 🚀 GitHub Pages를 통한 무료 웹 호스팅 배포 방법 (누구나 접속 가능하도록 만들기)

이 3종 파일을 GitHub에 올려 웹사이트로 호스팅하는 방법은 다음과 같습니다. 매우 간편하며 3분 안에 완료됩니다.

### Step 1. GitHub 저장소(Repository) 생성
1. [GitHub](https://github.com/) 로그인 후 우측 상단의 **`+`** 버튼을 누르고 **`New repository`**를 클릭합니다.
2. **Repository name**에 원하는 이름을 입력합니다 (예: `dept-finder`).
3. **Public**에 체크되어 있는지 확인합니다. (누구나 접속할 수 있도록 하려면 Public이어야 합니다.)
4. 최하단의 **`Create repository`** 버튼을 누릅니다.

### Step 2. 3종 파일 업로드
1. 생성된 저장소 화면 중간의 **`uploading an existing file`** 링크를 클릭합니다.
2. 이 폴더에 생성된 3개 파일(`index.html`, `style.css`, `app.js`)을 드래그 앤 드롭으로 모두 추가합니다.
3. 화면 하단의 **`Commit changes`** 버튼을 클릭하여 업로드를 완료합니다.

### Step 3. GitHub Pages 활성화 (웹 주소 만들기)
1. 저장소 상단 탭에서 **`Settings`** (설정) 아이콘을 클릭합니다.
2. 왼쪽 메뉴바에서 **`Pages`** 메뉴를 선택합니다.
3. **Build and deployment** 섹션 아래의 **Branch** 설정을 `None`에서 **`main`** (또는 `master`)으로 변경합니다.
4. 바로 옆의 폴더 설정을 `/ (root)` 상태 그대로 두고 **`Save`** 버튼을 클릭합니다.

### Step 4. 완성된 웹 주소로 접속하기
1. 페이지가 새로고침되며 상단에 잠시 후 **`Your site is live at https://[내-아이디].github.io/dept-finder/`** 형태의 웹 사이트 주소가 나타납니다. (약 30초~1분 소요)
2. 해당 주소를 클릭하면 전 세계 누구나 웹 브라우저를 통해 실시간으로 소관부서 추적 엔진을 사용할 수 있습니다!
